Multigraph Kernel – a run-time environment for model-based systems.

 

The Multigraph Kernel is a dynamic macro-dataflow execution environment that can execute on single processor or distributed parallel architectures. Dataflow control graphs can be built at run time, from user supplied functions, to be executed at the various nodes of the graph. A full set of API services is provided to control the run-time behavior of the synthesized applications.
